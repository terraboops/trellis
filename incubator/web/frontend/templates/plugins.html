▶ How plugins work with incubator agents

Plugins extend Claude Code with MCP servers, hooks, skills, and LSP servers. In the incubator, plugins are installed at the user level via Claude Code and are available to all agents.

Installing a plugin

From Claude Code, run:

/plugin install plugin-name@marketplace-name

Adding a marketplace

/plugin marketplace add owner/repo

Marketplace format

A marketplace is a repo with .claude-plugin/marketplace.json listing plugins.
